
StudioRich LoopFactory CSV Version

How to use:
1. Install dependencies (Terminal):
   pip install librosa soundfile numpy==1.23 pandas

2. Place your full-length tracks in 'input_audio/'.

3. Edit 'track_loop_list.csv' to list all the files you want processed.

4. Run:
   python batch_loop_csv.py

5. Results and loop file names will be saved in 'loop_results.csv'.

Auto-loop your catalog the smart way.
